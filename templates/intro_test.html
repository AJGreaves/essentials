Form

Forms have many usees:

  • Login and registration to secure websites
  • Customer feedback
  • Live chat
  • Booking events
  • Buying products online
  • Polls
  • Anything that requires the submission of user information

Forms collect input from the user. They are essential for the interaction between a user and a website or application. They contain input elements where a user can enter information, such as an email address or a password. You see and use forms all the time online.

A <form> element is composed of one or more child elements. Those elements can be text fields (single line or multiline), select boxes, buttons, checkboxes, radio buttons and many other elements used to collect data.

HTML Tables

HTML tables are used to organize data or information in columns and rows. Tables can be used for anything from displaying hospital patient data to cateloging Pokemon cards.

A HTML table uses 4 different types of element to construct a table:

  1. The <table> element is the parent element for the entire table, all the other elements nest inside it.
  2. The <tr> element, tr stands for Table Row and defines each row of the table, each row is nested directly inside the <table> element.
  3. The <th> element, th stands for Table Header. <th> is used to define the headings for the table, typically displayed in the top row.
  4. The <td> element, td stands for Table Data. Use <td> for all your table cells that contain the data to be displayed.
