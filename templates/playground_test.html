Now you have learned some more about html elements, let's play around with the heading, paragraph and topic break elements in the Playground. You may notice that the html, head and body blocks are not available this time. The playground allows us to work without these blocks, so they will not always appear. This is so we can focus on small parts of the structure and code at a time.

Instructions:

  1. Place a heading block onto the playground board.
  2. Add a text block to the heading block, and add the text “Welcome to Coders Coffeehouse!” to it.
  3. Connect a paragraph block underneath the heading block.
  4. Add a text block to the paragraph block, and add the text “Turning caffeine into code since 1999.” to it.
  5. Connect a topic break block underneath the paragraph block.

Check out the html output to see how the structure of what you have built is translated into code. Notice that the topic break element tag <hr> has no closing tag, that is because this element is a self-closing tag, because it creates a horizontal rule across the page, it doesn't have any content.
