Instructions

Now that you have learned about the different levels of headings, let's have a play with them in the playground. Feel free to change the heading levels around to see how this changes the code in the html, and the difference it makes to the font sizes in the preview.

  1. Place a heading block onto the playground board, change it's heading level to 2.
  2. Add the text Drinks Menu to the level 2 heading element.
  3. Connect a level 3 heading element under the level 2 element.
  4. Add the text Coffee to the level 3 heading element.
  5. Connect another level 3 heading element under the level 3 element.
  6. Add the text Tea to the level 3 heading element.

Try experimenting with changing the heading levels to see what changes in the output to the live preview.
