Let's try creating a very simple form using the playground.

Instructions:

  1. Place a form block into the playground board
  2. Nest a input block inside the form block
  3. Inside the form block, underneath the first input block, add a second input block. using the dropdown menu change it's selection from text to submit.

Notice how these elements now create an input field and submit button in the preview window. You may also notice that if you try to submit something in the form, the preview responds with a ?. This is because the form is not yet complete, and does not have what it needs to send the data somewhere.
